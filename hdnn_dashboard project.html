CNN-LSTM Architecture

Hybrid model with 30 layers: 18 convolutional, 12 pooling, 1 fully connected, 1 LSTM, softmax output.

CNN extracts local spatial features → LSTM captures temporal dependencies → Dense layers classify HD Positive / Negative.

Data split: 80% training · 20% testing
Normalization: Z-score (zero mean, unit variance)
Feature selection: Extra Tree Classifier (Gini)
Dropout: 20% per convolutional block
